## 💻 ホワイトハッカーが扱うネットワークノード一覧

ネットワーク上で情報資産を保持したり、重要な通信を担ったりするノードをカテゴリ別に分類して提案します。

### 1. サーバーとストレージ (情報資産の集中点) 💾

| ノード名 | アイコンのアイデア | 役割とハッキングの焦点 |
| :--- | :--- | :--- |
| **Webサーバー** (Web Server / HTTP) | 地球儀アイコン、`< >`タグ | 企業の公開情報やアプリケーションホスティング。SQLインジェクション、XSS、DDoS攻撃の対象。 |
| **データベースサーバー** (DB Server) | 円柱の積み重ねアイコン、鍵付きの宝箱 | 顧客データ、認証情報などの機密情報を保持。機密情報窃取（Data Exfiltration）の最終目標。 |
| **メールサーバー** (Mail Server / SMTP) | 封筒、@マーク | フィッシング、スプーフィング（なりすまし）の起点、情報の傍受。 |
| **ファイルサーバー** (File Server / FTP/SMB) | フォルダ、キャビネット | 機密文書、設定ファイルなどが置かれる。不正アクセス、ランサムウェアの対象。 |
| **認証サーバー** (Auth Server / LDAP/AD) | 盾、ユーザーアイコンが並ぶ | ユーザー認証とアクセス権管理の中核。資格情報窃取、ブルートフォース攻撃の対象。 |
| **DNSサーバー** | 住所録、矢印が上向きのブロック | ドメイン解決の中核。DNSスプーフィング、キャッシュ汚染の対象。 |

---

### 2. クライアントとエンドポイント (侵入の足がかり) 💻

| ノード名 | アイコンのアイデア | 役割とハッキングの焦点 |
| :--- | :--- | :--- |
| **ワークステーション** (Workstation) | デスクトップPC | 従業員の操作端末。フィッシングによる初期侵入、マルウェア感染の足がかり。 |
| **ノートPC** (Laptop) | 折りたたまれたPC | 持ち運び可能でセキュリティが手薄になりがち。社外からのアクセスポイント。 |
| **モバイルデバイス** (Mobile/Tablet) | スマートフォン | BYOD (私物端末の業務利用) のセキュリティホール、位置情報など。 |
| **IPカメラ** (CCTV/IoT Device) | カメラ | ネットワークの脆弱なエッジデバイス。踏み台、監視の妨害。 |

---

### 3. ネットワーク機器 (トラフィック制御) 🌐

| ノード名 | アイコンのアイデア | 役割とハッキングの焦点 |
| :--- | :--- | :--- |
| **ルーター** (Router) | 3本の矢印が交差する円 | 異なるネットワーク間の経路制御。トラフィックの傍受、ルーティング情報の改ざん。 |
| **スイッチ** (Switch) | 四角形に複数のポート | 同一ネットワーク内の通信の中継。MACフラッディング、ARPスプーフィングの対象。 |
| **ファイアウォール** (Firewall) | 赤い壁、盾と炎 | 外部からの不正アクセス防御の最前線。設定の誤り（誤設定）を見つけることが目標。 |
| **ロードバランサー** (Load Balancer) | 天秤、左右に分岐する矢印 | サーバーへのトラフィックを分散。サービス停止やセッションハイジャックの対象。 |
| **VPNゲートウェイ** | トンネル、鍵穴と鍵 | 外部からのセキュアなアクセスポイント。認証情報の盗難、VPNプロトコルの脆弱性。 |

---

### 4. セキュリティと監視 (防御の要素) 🛡️

| ノード名 | アイコンのアイデア | 役割とハッキングの焦点 |
| :--- | :--- | :--- |
| **IDS/IPS** (侵入検知/防御システム) | 監視カメラ、アラートサイン | 攻撃の検知と防御。検知ロジックの回避（Evading Detection）が目標。 |
| **SIEM** (Security Info & Event Mgmt) | 拡大鏡、レポート用紙 | ログの集中管理。ログの改ざん、システムの無効化。 |
| **ハニーポット** (Honeypot) | 蜂蜜の壺、囮（おとり） | 攻撃者を誘い込むための偽のサーバー。攻撃者はこれを本物と誤認して情報収集に利用しようとする。 |

これらのノードを組み合わせてマップを作成することで、プレイヤーはそれぞれのアイコンを見て、そのノードの重要性や攻撃方法を直感的に理解できるようになります。
